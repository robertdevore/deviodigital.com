BoostBox v2.0 Release: New Features, Improved Targeting, and More!

Sep 23, 2024 · Robert

We're excited to announce the release of BoostBox v2.0, packed with new features and enhancements that have been in the works for a long time. These updates aim to give you even more control over your popups and improve user engagement.

In this update, we've focused on improving the flexibility of targeting options, decreasing the plugins footprint on your page speed, and enhancing the overall functionality for managing multiple popups.

Ok, now here's a more detailed look at the key updates in BoostBox v2.0:

Enhanced Targeting Options

One of the most anticipated features of BoostBox v2.0 is the improved targeting functionality, which was up until this point lacking.

Now, you can display popups across various areas of your website like the home page, 404 page, search results, Custom Post Types (CPTs) and singular post/page targets more efficiently.

Whether you want to target specific post types or pages, BoostBox makes it easier to manage and customize where your popups appear.

  • New Target Options: We've added new target options directly to the display settings metabox. This means you can now control where your popups appear, from a central location rathern than a post-by-post basis. These target options give you more flexibility to create the perfect popup experience.
  • Multiple Popups Support: One of the biggest upgrades in this release is the support for handling multiple popups on the same page. This allows you to layer different popups, creating targeted campaigns with ease. We've updated the public scripts and tracking functionality to work seamlessly with multiple popups, so you can maximize conversions without worrying about conflicts.

New AJAX Functions

We've added two new AJAX functions to the admin panel, which allow for better post fetching and management within the BoostBox plugin.

These AJAX functions are integrated into class-boostbox-admin.php, streamlining the process of selecting and managing posts while you're creating popups in the admin area.

New Helper Functions for Developers

BoostBox v2.0 introduces four new helper functions, available in boostbox-helper-functions.php, to make it easier for developers to extend the functionality of BoostBox.

These helpers provide shortcuts for common tasks related to popup customization and management, making it simpler to add new features or adjust existing ones.

  • boostbox_popup_build_html
  • boostbox_popup_post_check
  • boostbox_detect_page_context
  • display_custom_post_type_select

Updated Design and User Interface

We've made several improvements to the user interface, particularly in the popup display settings:

  • Tabbed Display Settings: The display settings metabox has been optimized with new tabbed content for the Target settings, providing a cleaner, more organized interface for configuring popup behaviors. This makes it easier to toggle between display options, triggers, and targeting in a streamlined manner.
  • CSS Optimizations: The CSS for BoostBox has been refined to only enqueue styles if popups are present on the page. This helps to reduce the plugin's footprint, improving page load times and overall site performance.
  • Admin JS Updates: We've also updated the admin JS so that it only loads when you're on the popup edit screen. This reduces unnecessary script loading, keeping the admin panel lightweight and responsive.

Popup Management and Flexibility

With BoostBox v2.0, managing individual popups in WordPress is easier than ever:

  • Disable Popups on Specific Pages: A new checkbox option has been added to the popup settings metabox, allowing you to disable popups on a page-by-page basis. This is perfect for times when you want full control over where specific popups are displayed or suppressed.
  • Popup HTML Rendering: We've moved the rendering of popup HTML to wp_footer, instead of using template_redirect. This small but significant change ensures that popups are loaded in the correct order, reducing conflicts and improving overall compatibility with themes and other plugins.

General Code Cleanup and Performance Enhancements

BoostBox v2.0 also includes general code cleanup across multiple files. These optimizations ensure the plugin runs smoothly and efficiently, without adding unnecessary bloat to your site.

How to Upgrade

To upgrade to BoostBox v2.0, simply go to your WordPress dashboard and update the plugin from there.

If you're installing BoostBox for the first time, you can find it by searching "BoostBox" in the WordPress plugin repository or by downloading it directly from your account.

BoostBox v2.0 makes it easier to create targeted, high-converting popups with the tools and triggers you need to engage your audience effectively.

Whether you're a developer looking to extend its functionality or a marketer building campaigns, this release has something for everyone.

Start exploring the new BoostBox today and elevate your popup game!
