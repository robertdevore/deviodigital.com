Delivery Drivers for WooCommerce v3.4.0 Release Notes

Jun 13, 2023 · Robert

We are excited to announce the release of Delivery Drivers for WooCommerce version 3.4.0!

A long time in the making, this update brings several bug fixes, improvements, and general code cleanup to enhance the functionality and performance of the plugin.

These changes lay the foundation for version 4.0.0 which is already being actively worked on ü§òüíØ

In this blog post, we will outline the key changes introduced in the 3.4.0 version.

Bug Fixes

  • Fixed an issue in admin/ddwc-metaboxes.php where the driver metabox was not saving properly when saving the order.
  • Resolved a bug in admin/ddwc-metaboxes.php that caused the order notes and status changes in the driver metabox to malfunction when saving.
  • Fixed a broken redirect problem in admin/ddwc-functions.php that occurred when the driver changed the order status.
  • Addressed an issue in delivery-drivers-for-woocommerce.php where the esc string was incorrectly outputting HTML instead of text.

Improved Functionality

  • Updated the usage of wc_get_order() to utilize new WC_Order() in admin/ddwc-functions.php. This improvement ensures compatibility and consistency with the latest changes in WooCommerce.

Localization Enhancements

  • Updated text strings for localization to facilitate translations. These updates can be found in the languages folder of the plugin. By improving localization support, we aim to make Delivery Drivers for WooCommerce more accessible to users worldwide.

Code Cleanup

  • Conducted general code cleanup throughout multiple files in the plugin. By removing redundant code, optimizing existing functions, and enhancing the overall structure, we have improved the maintainability and performance of the plugin.

We encourage all users to update to Delivery Drivers for WooCommerce version 3.4.0 to take advantage of these important changes.

You can download the latest version from the official WordPress plugin repository or update directly from your WordPress dashboard.

If you encounter any issues or have any questions, please reach out to our support team.
