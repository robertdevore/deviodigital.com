Delivery Fees for WooCommerce v1.6.0 Release Notes

Jun 13, 2023 · Robert

We are pleased to announce the release of Delivery Fees for WooCommerce version 1.6.0!

This update brings important improvements to enhance the compatibility and localization capabilities of the plugin.

In this blog post, we're going to provide an overview of the key changes introduced in this version.

PHP 8.0 Compatibility

To ensure seamless compatibility with the latest PHP version, we have updated the code in multiple files throughout the plugin.

The Delivery Fees for WooCommerce plugin is now fully compatible with PHP 8.0, offering improved performance, security, and functionality.

With this update, you can confidently run the plugin on the latest PHP environment.

Localization Enhancements

We understand the importance of providing a localized experience for our users worldwide. In version 1.6.0, we have updated the text strings used in the plugin to facilitate translations.

These updates can be found in the languages folder, allowing you to easily translate the plugin into different languages to better serve your customers.

By improving localization support, we aim to make Delivery Fees for WooCommerce more accessible and user-friendly for a global audience.

Whether you're targeting local markets or expanding your business internationally, our plugin now provides the tools to cater to diverse language preferences.

Update ASAP

We encourage all users to update to Delivery Fees for WooCommerce version 1.6.0 to take advantage of these important changes.

You can download the latest version from the official WordPress plugin repository or update directly from your WordPress dashboard.

If you have any questions or encounter any issues, please don't hesitate to reach out to our support team.

We value your feedback and are committed to continuously improving the Delivery Fees for WooCommerce plugin to meet your needs.
