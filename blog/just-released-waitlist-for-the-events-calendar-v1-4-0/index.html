If you're an event organizer using The Events Calendar, this update ensures that your waitlist functionality runs smoothly while providing a better experience for attendees.

üêû Key Bug Fix: Multi-Event Tickets Now Display Correctly

In previous versions, there was an issue where if one ticket type** sold out, **all other ticket types would be hidden -- even if they were still available.

This caused confusion for both event organizers and attendees, sometimes preventing users from securing tickets to an event that wasn't actually sold out.

We've fixed this issue in v1.4.0, ensuring that only the sold-out ticket types are hidden while keeping available tickets visible.

This means a smoother purchasing experience and fewer headaches for event organizers who rely on waitlists for high-demand events.
