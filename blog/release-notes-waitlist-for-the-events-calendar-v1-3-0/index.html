Release Notes: Waitlist for The Events Calendar v1.3.0

Sep 09, 2024 · Robert

We are excited to announce the release of version 1.3.0 of Waitlist for The Events Calendar. This update brings several improvements and optimizations designed to enhance your experience and streamline your workflow.

What's New in Version 1.3.0?

1. Waitlist Count and Quick Access in Events Table

Managing your event waitlists just got easier! The Events table includes a view of the waitlist count, providing you with a quick overview of the number of people on the waitlist for each event. Additionally, we just added a link to view the full waitlist has been added, making navigation more seamless and efficient. You can find this update in the file admin/class-waitlist-for-the-events-calendar-admin.php.

2. Improved Success Message Alignment and Style

We've polished the success message users receive when they join the waitlist. The alignment and styling have been refined to provide a cleaner, more professional look as well as an updated text string for better readability. Check out the updates in public/css/waitlist-for-the-events-calendar-public.css for a better visual experience.

3. Modernized Array Syntax

In this release, we've updated array syntax to shorthand throughout multiple files. This change ensures the plugin adheres to modern coding standards, making it easier to read and maintain.

4. Localization Improvements

To make Waitlist for The Events Calendar more accessible to a global audience, we have updated the text strings throughout various files for better localization. This will make it easier to translate and use the plugin in different languages. Feel free to contact us if there's a specific language you'd like to see the plugin translated into!

5. General Code Cleanup

We've also taken the opportunity to perform general code cleanup across multiple files. This ongoing effort improves performance, makes the codebase more efficient, and lays the groundwork for future updates.

Update Today!

To take advantage of these improvements, be sure to update to version 1.3.0. You can download the latest release from the Account page. We're always striving to make Waitlist for The Events Calendar better, and we appreciate your support and feedback.

Stay tuned for more updates and features in the future!
